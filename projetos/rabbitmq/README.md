# rabbirmq-nodejs

Projeto Nodejs que demonstra enviar e receber mensagens do RabbitMQ.

### Para testar
Inicie um servidor RabbitMQ. Recomendo um docker.
cd rabbirmq-nodejs
npm install
node Send.js -> para enviar
node Consumer.js -> para consumir


### Tecnologias utilizadas:
* Nodejs
* RabbitMQ
